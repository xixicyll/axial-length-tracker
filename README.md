# Axial Length Tracker

## Overview
The **Axial Length Tracker** web app is designed to facilitate eye care professionals in tracking and managing the axial lengths of patients' eyes over time. This application provides an intuitive interface for users to record, analyze, and visualize axial length data efficiently.

## Key Features
- **User Authentication**: Secure login for health professionals.
- **Data Entry**: Simple forms to input axial length measurements, associated patient information, and dates.
- **Data Visualization**: Graphs and charts to visualize changes in axial lengths over time for individual patients.
- **Reports Generation**: Ability to generate reports summarizing patient data and axial length measurements.

## Technologies Used
- **Frontend**: React.js
- **Backend**: Node.js with Express
- **Database**: MongoDB
- **Visualization Library**: Chart.js

## Installation
To set up the Axial Length Tracker locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/xixicyll/axial-length-tracker.git
   ```
2. Navigate to the project directory:
   ```bash
   cd axial-length-tracker
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
After successfully starting the application, visit `http://localhost:3000` to access the Axial Length Tracker. From there, you can log in to your account and start tracking axial lengths.
